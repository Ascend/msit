# Optimize Command #

## Introduction ##

Uses the specified knowledge base to optimize the specified ONNX model.

```
msit debug surgeon optimize [OPTIONS] [REQUIRED]
```

optimize can be abbreviated as opt.

This parameter is valid only when the infer-test option is enabled. --loop: indicates the number of inference times when the inference speed is tested. This parameter is valid only when the infer-test option is enabled. The default value is 100. --threshold: indicates the inference speed increase threshold. The knowledge base is used only when the improvement caused by the optimization of the knowledge base exceeds the value of this parameter. The value can be negative. A negative value indicates that the negative optimization is accepted. The default value is 0. That is, only optimizations with improved inference performance are accepted by default. ## Running an Example ##

```
msit debug surgeon optimize --input=aasist_bs1_ori.onnx --output-file=aasist_bs1_ori_out.onnx
```
