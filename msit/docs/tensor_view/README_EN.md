# Msit Tensor View User Guide

## Brief Introduction

The tensor-view tool provides an interface for viewing tensors. The source data is the bin file generated after dump. Chain slicing and transposing operations are carried out. By default, the statistics and tensor.shape are printed each time. You can print the tensor itself and save it to a file. The file format can be the torch or ATB format (same as the file generated by dump).

Windows is not supported currently.

## Environment preparation

 * Installed`python >= 3.7`environment
 * **Install the msit tool. The installation reference document is as follows:**[Installing the msit tool](https://gitcode.com/Ascend/msit/blob/master/msit/docs/install/README.md)    
 * **Data preparation**
    
     * Tensor parameter file path (.bin/.pth). The ATB format (.bin) and Torch format (.pth) are supported,
 * Example command
    
    ```shell
    #Enter the ATB file and apply continuous slice, transpose, slice, and transpose. The operations is for demonstration only. This example will fail because the first transpose does not comply with the rules (0, 2, 1, 4).
    msit tensor-view --bin intensor0.bin --operations "[1:2, ..., 3::2];(0,2,1,4);[1:3];(2,0,1)" --output tmp_view/output_view.bin
    ```
    
    ```shell
    #Input PTH, output ATB format file, use abbreviated parameter name.
    msit tensor-view -b rand.pth -op "[1];(2,0,1)" -o out/processed_rand.bin
    ```

## Parameter Description

| Parameter name    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        | Mandatory. |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- |
| --bin, -b         | Tensor parameter file path, which can be in ATB (.bin) or Torch (.pth) format.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     | Yes        |
| --print, -p       | Indicates whether to print Tensors on the console. By default, Tensors are not printed. Note that the print operation follows operations.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          | No.        |
| --operations, -op | Character string for the Tensor slicing and permute operations, which must be enclosed in double quotation marks. Segmentation and slicing operations need to be wrapped in \[\] and transpose operations need to be wrapped in (). These operation strings are applied to the tensor in sequence. During the execution, the system checks whether the slicing and transpose strings are valid, whether the slice index exceeds the boundary, and whether the transpose strings correspond to dimensions. Note that the slicing supports standard slicing, digital indexing, and Ellipsis. The three types of slicing can be combined in an extended slicing. True/False slicing is not supported. | No.        |
| --output, -o      | Path for storing the processed Tensor. There is no default value. If the file name extension is .bin, the file is saved in ATB format. If the file name extension is .pth, the file name extension is torch.save. If the file name extension is omitted, the file name format is the same as the \[--bin\] format.                                                                                                                                                                                                                                                                                                                                                                                 | No.        |
| -h, --help        | Help information about the tool                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | No.        |
