# 安装

## 1 依赖版本

| 依赖软件名 | 是否必选 | 版本 |
|------|-----|-----|
| Python | 是 | 3.7, 3.8, 3.9, 3.10, 3.11 |
| [PyTorch (CPU)](https://pytorch.org/) | 否 | 2.1, 2.2, 2.3, 2.4，对应的 Python 版本最低为 3.8 |
| [PyTorch (NPU)](https://gitee.com/ascend/pytorch) | 否 | 2.1, 2.2, 2.3，对应的 Python 版本最低为 3.8 |
| [TensorFlow]((https://github.com/tensorflow/tensorflow/releases/tag/v2.6.5)) | 否 | 仅支持 2.6.5 版本，对应的 Python 版本为 3.7~3.9 |

用户可以根据使用场景自行安装依赖。

## 2 安装 msIT

### 2.1 从源码安装

```sh
git clone https://gitee.com/ascend/msit.git -b reboot
cd msit/inference_tools

pip install setuptools wheel

python setup.py bdist_wheel
cd ./dist
pip install msit*.whl
```
